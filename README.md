# Flask Eindproject

Deze website is gemaakt met Flask en gebruikt templates in de map `templates/`.

## Installatie

1. Open een terminal in de projectmap.
2. Installeer de vereiste Python-pakketten:

```bash
python3 -m pip install -r requirements.txt
```

## Starten

Start de Flask-app vanuit de hoofdmap van het project:

```bash
python3 main.py
```

Open daarna in je browser:

```
http://127.0.0.1:5001/
```

> Belangrijk: open nooit de repositorymap direct als een statische server of met een bestandspad. De website moet via Flask worden gestart.
